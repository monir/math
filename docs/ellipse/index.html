Two foci, one cup runneth over again:
the ellipse perimeter, cracked harder than ever

UNDERWOOD — Undivided Numerical · Direct Elliptic Residue · Wave-Orthogonal Operator Decomposition. Chebyshev polynomials in $u = 2\xi - 1$ plus explicit log companions $\xi^{2+j}\!\ln(1/\xi)$. No tower, no gate, no exponentials, no iteration, no denominator.

Three log functions written into a Chebyshev basis give 3.3 parts per trillion at 13 stored constants — half the parameter budget of the author's earlier SRIRACHA record (7 ppt at ∼25 parameters), for twice the accuracy. With one more log mode the formula saturates machine precision (3 × 10−13 relative) at 14 constants.

The five-rung ladder, fully written out

The same arc the paper opens with: easy handy → old mid → new mid → old winner → new champ. Each rung is a closed-form approximation; lower rungs are more recent and more accurate.

Rung 0 · Easy Handy · Ramanujan I (1914)

0 free parameters · peak error ~63 ppm

$$P_{\text{R1}}(a,b) \;=\; \pi\,\Bigl[\,3(a+b) \;-\; \sqrt{(3a+b)(a+3b)}\,\Bigr].$$
Rung 1 · Old Mid · Ramanujan II (1914)

0 free parameters · peak error ~7 ppm

$$P_{\text{R2}}(a,b) \;=\; \pi(a+b)\,\Biggl[1 + \frac{3h}{10 + \sqrt{4 - 3h}}\Biggr], \quad h = \Bigl(\tfrac{a-b}{a+b}\Bigr)^{\!2}.$$
Rung 2 · New Mid · Ayala-Raggi–Rendón-Marín (2025–26)

3–15 free parameters · 0.5 ppm → 0.0216 ppm at the top

$$P_{\text{AR}}(a,b) \;=\; \pi(a+b)\Bigl[1 + \tfrac{3h}{10+\sqrt{4-3h}}\Bigr] \,\cdot\, \Bigl(1 + \sum_{k=1}^{N}\alpha_k\,h^{\beta_k}\,e^{-\gamma_k(1-h)}\Bigr)^{\!-1}.$$

Single-exponential corrections on top of Ramanujan II, fit by constrained least squares. Strongest external competitor in the literature.

Rung 3 · Old Winner · SRIRACHA R2/3exp V23 (Mamoun, Feb 2026)

8 stored constants · peak error 0.083 ppm

$$P_{\text{SRIRACHA}}(a,b) \;=\; \frac{P_{\text{R2}}(a,b)}{\,1 \;-\; h^{7/3}\,\bigl[A\,e^{-\lambda x} + B\,e^{-(81/25)\lambda x} + C\,e^{-15\lambda x}\bigr]\,}$$

With $x = 1-h$. The four "rate-shape" constants $(q,m,n,T)$ fall out of the continued fraction $\pi=[3;7,15,1,292,\ldots]$ and are not fitted; the three free constants are $\lambda$, $R=B/A$, $S=C/A$; amplitude $A$ is forced by the endpoint pin $A(1+R+S) = 1 - 7\pi/22$. Eight stored numbers, three free parameters.

Rung 4 · New Champ · UNDERWOOD F4 (this work)

14 stored constants · peak error 3 × 10−13 relative

$$P_{\text{UNDERWOOD-F4}}(a,b) \;=\; \frac{a+b}{2}\,\Bigl[\, \sum_{k=0}^{8} c_k\,T_k(u) \;+\; \sum_{j=0}^{4} d_j\,\xi^{2+j}\,\ln(1/\xi)\,\Bigr]$$

With $h=\bigl(\tfrac{a-b}{a+b}\bigr)^{\!2}$, $\xi = 1-h$, $u = 2\xi - 1$, and $T_k$ the Chebyshev polynomial of the first kind. All 14 coefficients are listed exactly in the paper’s cookbook and emitted by scripts/underwood/formula_renderer_dual_v1.py so the paper can never drift from the running code.

python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
